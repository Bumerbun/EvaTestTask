
## Установка зависимостей 

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Запуск сервера

Запуск сервера происходит на `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```
